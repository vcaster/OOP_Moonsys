There is something which you need to know about XAMP. 

Currently, there is XAMP 3.2.2 is the latest version out there which supports PHP 7.1. 

As this course has designed for a complete beginners also for those who don't have a programming background. That's why this course uses XAMP 3.2.1 earlier version  with PHP 5.6. 

All the PHP code is fully supported in both version but for database its bit different. XAMP with PHP 5.6 can let you work with database (SQL) by using mysql, however, XAMP with PHP 7.1 can only let you work with database (SQL) by using mysqli not mysql.  And mysqli syntax is bit difficult for beginners. 

That is why this whole course uses XAMP with PHP 5.6 with mysql not mysqli.

I will recommend you to use PHP 5.6 with this course so that you will learn and wont get stuck in tricky mysqli syntax. 

After completing this course you can easily work and shift later in future on mysqli with little work.
